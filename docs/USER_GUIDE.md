## 快速入门

### 访问网站

1. 打开浏览器访问网站地址
2. 首次访问会显示欢迎引导
3. 点击"下一步"了解各功能模块
4. 点击"跳过"可跳过引导

### 安装到手机

**iOS设备：**
1. 在Safari中打开网站
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

**Android设备：**
1. 在Chrome中打开网站
2. 点击右上角菜单
3. 选择"添加到主屏幕"
4. 点击"安装"

安装后可像App一样使用，支持离线访问。

## 账号系统

### 注册账号

1. 点击右上角"登录"按钮
2. 切换到"注册"标签
3. 填写信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 密码（必填）
   - 身份（学生/教师/家长/志愿者）
4. 点击"注册"

### 登录账号

1. 点击"登录"按钮
2. 输入用户名和密码
3. 点击"登录"

### 退出登录

1. 点击右上角头像
2. 选择"退出登录"

## 社区功能

### 发布帖子

1. 进入"社区"页面
2. 在顶部编辑框输入内容
3. 选择话题分类：
   - 高考备考
   - 学习经验
   - 资源分享
   - 问答求助
   - 教育平权
4. 点击"发布"

### 浏览帖子

1. 在社区页面浏览帖子列表
2. 使用筛选器：
   - 最新 - 按时间排序
   - 热门 - 按点赞数排序
   - 关注 - 关注用户的帖子
3. 点击帖子查看详情

### 互动操作

| 操作 | 说明 |
|-----|------|
| 点赞 | 点击👍图标 |
| 收藏 | 点击⭐图标 |
| 评论 | 在评论区输入内容 |
| 分享 | 点击分享图标 |

### 话题分类

- **高考备考** - 高考相关经验、策略
- **学习经验** - 学习方法、技巧分享
- **资源分享** - 学习资料、工具推荐
- **问答求助** - 问题咨询、求助
- **教育平权** - 教育公平相关讨论

## 智能体使用

### 创建智能体

1. 进入"智能体"页面
2. 点击"+ 创建智能体"
3. 填写信息：
   - 名称
   - 类型
   - 描述
   - 标签
4. 选择头像
5. 点击"创建"

### 智能体类型

| 类型 | 功能 |
|-----|------|
| 学习辅导 | 学科知识讲解、作业辅导 |
| 问答助手 | 快速回答各类问题 |
| 学习规划 | 制定学习计划、进度管理 |
| 心理辅导 | 缓解压力、情绪支持 |
| 自定义 | 自由定义功能 |

### 与智能体对话

1. 点击智能体卡片
2. 在输入框输入问题
3. 按回车或点击"发送"
4. 智能体会流式返回回答

### 对话技巧

- 明确描述问题
- 提供必要的上下文
- 一次问一个问题
- 可以追问细节

## 高考资源

### 浏览资源

1. 进入"高考资源"页面
2. 使用筛选器：
   - 科目：语文/数学/英语/物理/化学/生物/政治/历史/地理
   - 类型：真题/模拟题/复习资料/知识点
   - 地区：全国/各省

### 获取资源

1. 点击资源卡片
2. 查看资源描述
3. 点击"获取资源"
4. 按提示下载或查看

## 常见问题

### Q: 如何离线使用？

A: 网站是PWA应用，首次访问后会自动缓存。之后即使断网也能访问已缓存的内容。

### Q: 数据会丢失吗？

A: 数据存储在本地IndexedDB中，不会丢失。建议定期导出备份。

### Q: 如何切换深色模式？

A: 点击右上角月亮图标，或在设置中选择主题模式。

### Q: 智能体回答不准确怎么办？

A: 智能体基于预设规则回答，建议：
- 描述更具体
- 提供更多上下文
- 尝试不同表述

### Q: 如何分享内容？

A: 点击分享图标，选择平台：
- 微信
- 微博
- QQ
- 复制链接

### Q: 忘记密码怎么办？

A: 本地存储模式，重新注册即可。建议导出数据后再操作。

### Q: 如何联系管理员？

A: 在页脚找到"联系我们"，发送邮件至 contact@eduequity.com
