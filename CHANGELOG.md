# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2024-01-15

### 新增

#### 核心功能
- 完整的社区系统（帖子发布、评论、点赞、收藏）
- 智能体创建系统（5种类型、Markdown渲染、流式响应）
- 高考资源中心（按科目/类型/地区筛选）
- 教育平权模块（使命介绍、数据统计、志愿活动）
- 用户系统（登录/注册、个人中心、设置）

#### PWA特性
- Service Worker离线支持
- 可安装到桌面
- 推送通知框架
- 后台同步

#### 性能优化
- 骨架屏加载
- 虚拟滚动
- 图片懒加载
- 资源预加载

#### 用户体验
- 深色模式
- 主题定制（5种预设+自定义颜色）
- 响应式设计
- 手势操作（滑动、长按、双击、捏合）
- 快捷键系统（12个全局快捷键）
- 用户引导系统

#### 数据可视化
- 学习统计图表
- 时间分布柱状图
- 科目进度展示

#### 国际化
- 中英文双语支持
- 自动语言检测

#### 智能功能
- 推荐引擎
- 拼音搜索
- 模糊匹配

### 技术特性

#### 架构
- 事件驱动架构（EventBus）
- 集中式状态管理（Store）
- 单页应用路由（Router）
- 模块化组件系统

#### 存储
- IndexedDB持久化存储
- LocalStorage配置存储
- 离线队列支持
- 数据导入导出

#### 样式
- CSS变量主题系统
- 响应式断点系统
- 动画系统
- 渐变主题

## 贡献指南

### 提交规范

```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式
refactor: 重构
perf: 性能优化
test: 测试
chore: 构建/工具
```

### 示例

```
feat: 添加深色模式支持
fix: 修复路由导航问题
docs: 更新部署文档
style: 优化代码格式
refactor: 重构状态管理
perf: 优化首屏加载
```
